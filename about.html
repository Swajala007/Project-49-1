Python:

  • Python is an interpreted high-level general-purpose programming language. Its design philosophy emphasizes code readability with its use of significant indentation. Its language constructs as well as its object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.
  • Developer: Python Software Foundation
  • First appeared: February 20, 1991; 30 years ago
  • Designed by: Guido van Rossum
  • OS: Windows, Linux/UNIX, macOS and more
  • Stable release: 3.10.0 / 4 October 2021; 36 days ago
  • Paradigm: Multi-paradigm: object-oriented, procedural (imperative), functional, structured, reflective
  • Typing discipline: Duck, dynamic, strong typing; gradual (since 3.5, but ignored in CPython)

Code.org:

  • Code.org is a non-profit organization and eponymous website led by Hadi and Ali Partovi that aims to encourage people, particularly school students in the United States, to learn computer science.
  • Founded: January 2013
  • Headquarters: Seattle, Washington
  • Purpose: Improving American computer education
  • Founders: Hadi Partovi, Ali Partovi

P5.js:

  • p5.js is a JavaScript library for creative coding, with a focus on making coding accessible and inclusive for artists, designers, educators, beginners, and anyone else! p5.js is free and open-source because we believe software, and the tools to learn it, should be accessible to everyone.

Php:

  • PHP is a general-purpose scripting language geared towards web development. It was originally created by Danish-Canadian programmer Rasmus Lerdorf in 1994. The PHP reference implementation is now produced by The PHP Group.

BootStrap:

  • Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.
  • License: MIT License (Apache License 2.0 prior to 3.1.0)
  • Platform: Web platform
  • Written in: HTML, CSS, Less (v3), Sass (v4) and JavaScript
  • Developer(s): Bootstrap Core Team
  • Repository: Bootstrap Repository
  • Initial release: August 19, 2011; 10 years ago
  • Stable release: 5.1.3 / 9 October 2021; 30 days ago

ReactNative:

  • React Native is an open-source UI software framework created by Meta Platforms, Inc. It is used to develop applications for Android, Android TV, iOS, macOS, tvOS, Web, Windows and UWP by enabling developers to use the React framework along with native platform capabilities.
  • Developer(s): Facebook and community
  • Platform: Android, Android TV, iOS, macOS, tvOS, Web, Windows, UWP, and VR
  • Initial release: March 26, 2015; 6 years ago
  • License: MIT License
  • Stable release: 0.65.1 / August 19, 2021; 2 months ago
  • Written in: JavaScript, Java, C++, Objective-C, Objective-C++, Python
